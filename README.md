# nixconfig

Personal NixOS configuration using flake-parts and Home Manager.

## Structure

```
nixconfig/
├── flake.nix                        # Entry point — declares inputs, delegates to modules/
├── flake.lock                       # Pinned dependency versions (commit this)
└── modules/
    ├── parts.nix                    # Flake-parts boilerplate (supported systems)
    ├── features/
    │   └── niri.nix                 # Optional feature modules (enable per-host)
    └── hosts/
        └── janma/
            ├── default.nix          # Wires the host together (NixOS + Home Manager)
            ├── configuration.nix    # System-level config (services, boot, networking)
            ├── hardware.nix         # Hardware-specific config (generated by nixos-generate-config)
            └── _home/
                └── home.nix         # User-level config (packages, dotfiles, shell)
```

### Two layers

| Layer | File | Manages |
|---|---|---|
| System | `configuration.nix` | Boot, networking, services, system-wide programs |
| User | `_home/home.nix` | User packages, dotfiles, shell, editor config |

### Why `_home/` has an underscore

`import-tree` auto-imports every `.nix` file under `modules/` as a flake-parts module.
Files under directories prefixed with `_` are ignored by import-tree, so `home.nix`
can live next to the host it belongs to without being incorrectly picked up.

### Adding a new host

1. Create `modules/hosts/<hostname>/`
2. Copy `default.nix`, `configuration.nix`, and `hardware.nix` from an existing host
3. Replace the hardware config with output from `nixos-generate-config`
4. Add a `_home/home.nix` for the user config

### Adding a new feature module

Create `modules/features/<name>.nix` — it will be auto-imported by import-tree.
Expose it as a `nixosModule` and import it in the relevant host's `default.nix`.

## Commands

### Rebuild and switch to the new config
```bash
sudo nixos-rebuild switch --flake .#janma
```

### Rebuild without switching (test first)
```bash
sudo nixos-rebuild test --flake .#janma
```

### Build without applying (check for errors)
```bash
nixos-rebuild build --flake .#janma
```

### Update all flake inputs
```bash
nix flake update
```

### Update a single input
```bash
nix flake update nixpkgs
nix flake update home-manager
```

### Run garbage collection manually
```bash
nix-collect-garbage --delete-older-than 14d
```

### List generations
```bash
nixos-rebuild list-generations
```

### Roll back to the previous generation
```bash
sudo nixos-rebuild switch --rollback
```
